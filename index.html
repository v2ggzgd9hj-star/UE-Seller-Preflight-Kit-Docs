UE Seller Preflight Kit Documentation

UE Seller Preflight Kit is an independent Unreal Engine editor plugin from Toolsmith Studio. It runs local checks against a project or plugin folder, shows actionable findings in the editor, and exports Markdown, CSV, and JSON reports for cleanup tracking.

This plugin is not affiliated with Epic Games, Unreal Engine, or Fab. It does not guarantee marketplace approval and does not replace the official requirements.

Safety And Privacy

Normal scan and export are non-destructive. The plugin does not upload project files, call AI services, use cloud services, or intentionally collect telemetry. Redirectors are reported in V1; use Unreal Editor's normal redirector cleanup workflow manually if a report identifies redirectors.
