# Support

Support is for UE Seller Preflight Kit installation, scanning, export behavior, documentation, and bug reports. It is not support for guaranteeing Fab approval.

UE Seller Preflight Kit is independent and unofficial. It is not affiliated with Epic Games, Unreal Engine, or Fab.

Publisher: Toolsmith Studio

Support email: toolsmithstudio@proton.me

Version: 1.0.0

Tested environment: Windows / Win64 with Unreal Engine 5.7.4

Toolsmith Studio creates practical Unreal Engine editor tools for creators, technical artists, and small teams. The focus is on local workflows, clear documentation, safe tooling, and repeatable checks that reduce friction during development and packaging.

Support Covers

  • Plugin installation questions.
  • Plugin enable/load problems.
  • Scan errors.
  • Unexpected result wording.
  • Export failures.
  • JSON/CSV/Markdown report issues.
  • Documentation template generation issues.
  • Reproducible crashes related to this plugin.

Support Does Not Cover

  • Approval promises.
  • Fab account, tax, payout, or listing administration.
  • Legal, trademark, copyright, or licensing advice.
  • Rewriting your product documentation.
  • Fixing all project content issues for you.
  • Automatic asset cleanup.
  • Unreal Engine issues unrelated to this plugin.

Information To Provide

Include as much of this as possible:

  • Unreal Engine version.
  • UE Seller Preflight Kit version.
  • Operating system.
  • Whether the target is Current Project, This Plugin, or Manual Path.
  • Target type: code plugin, content project, or other.
  • Exported Markdown, CSV, or JSON report if available.
  • Screenshot of the plugin window.
  • Any visible status/error text.
  • Steps to reproduce.
  • Whether the issue happens in a clean test project.

Privacy And Local Data

The plugin runs locally. It does not upload project files, call AI services, use cloud services, or intentionally collect telemetry. Only send reports or screenshots to support if you choose to share them.
